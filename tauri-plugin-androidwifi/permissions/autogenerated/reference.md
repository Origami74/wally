## Default Permission

Default permissions for the plugin

- `allow-get-wifi-details`
- `allow-get-mac-address`
- `allow-get-current-wifi-details`
- `allow-connect-wifi`
- `allow-mark-captive-portal-dismissed`
- `allow-register-listener`
